# <img src="logo.svg" width="64" alt="Atone" /> 

> Atone is a Linux init-like process supervisor.

## Features

* Orphan-zombie process reaping
* Services spawn
* Single-service mode
* Multi-service mode
* Service restart

### Future features

* Service schedule
